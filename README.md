# Змейка
Классическая игра «Змейка» на Python с использованием Pygame
# Особенности
* Постепенный разгон скорости по мере роста змейки (с ограничением)
* Пауза во время игры
* Сохранение и подсчёт топ 5 рекордов
# Требования
* Python 3.8+
* Pygame
# Установка зависимости
pip install pygame
# Запуск
открываем файл snake
# Управление
* WASD/стрелки - движение змейки
* ESC - пауза
* R - перезапуск (после Game Over)
* Q - выйти из игры (после Game Over)
* Alt + F4/нажать на крестик в окне - закрыть игру
# Игровая механика змейки
* Развернуться на 180° нельзя - змейка не может мгновенно пойти в противоположную сторону от себя
* Поле это сетка 25 на 25
* Скорость растёт на 1 за каждые 3 съеденные еды, начиная с 5 и до максимума 16
* Таблица топ 5
# Сохранение рекордов
Рекорды хранятся в файле scores.json, который создаётся автоматически рядом со скриптом игры в той же папке
# Возможные доработки
* Отвязать скорость змейки от fps
* Создать возможность делать никнеймы для отображения в таблице рекордов (но без онлайн функций)
* Возможность выбрать размер поля
* Добавить возможность сделать лимит на то сколько максимум может находится еды для змейки на поле
